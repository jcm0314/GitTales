GitTales

백준처럼 풀고, AI에게 리뷰받고, 캐릭터를 키우는
실무 과제테스트 연습 & 1:1 피드백 플랫폼

1. 깃허브 실무 과제 제출 ➔ 2. AI 자동 코드 리뷰 및 채점 ➔ 3. 내 실수 기반 맞춤형 학습(퀴즈)
The Problem

알고리즘 코딩테스트의 종말,
준비된 대안이 부재한 개발자 채용 시장

🧠
코테 변별력 상실
AI 자동완성(Copilot, ChatGPT)의 일상화로 알고리즘 코테는 변별력을 잃어 실제 코드를 구현하는 과제테스트 선호도 급증.
🧱
피드백 차단 (주니어)
과제테스트 탈락 시 "내 코드가 왜 떨어졌고 어떤 부분이 부족한지" 알려주는 연습/피드백 플랫폼 전무.
💸
기업 및 교육기관의 막대한 채점 피로
수십 명의 복잡한 프로젝트 코드를 수동 채점해야 하여 시니어 개발자 리소스가 낭비되거나 부실한 피드백 전달.
Our Solution

실무 과제 제출부터 개인 취약점 극복까지,
AI 기반의 선순환 훈련 루프

  • 1단계: 실무 과제 기출문제 Clone
    유저는 웹에서 제공하는 과제 보일러플레이트를 복사하여 개발 후 GitHub PR 제출.
  • 2단계: AI 자동 코드 리뷰 및 정량 채점
    제출 즉시 AI가 코드 퀄리티 분석, 카테고리별 채점 및 깃허브 라인별 피드백 코멘트 실시간 작성.
  • 3단계: 실수 기반 RPG 게이미피케이션 학습
    내 코멘트 기반 맞춤형 오답 퀴즈(객관식/Explain-Back)를 풀어 스텟(LTI 지수) 및 캐릭터 레벨업.
Differentiation & Fit

시중 B2B 평가 도구들과 다른
GitTales만의 독보적 경쟁력

시중 AI 리뷰어 (B2B)
단순 정답 코드 즉시 제안 ➔ 개발자의 기계적 수용으로 생각하는 힘 상실 (학습 효과 X).
시중 채용 툴 (B2B)
기업 전용 스크리닝 용도 ➔ 일반 취준생은 연습용 가입 및 피드백 획득이 원천 불가능.
GitTales (B2C 오픈)
취준생 누구나 무료 실습 가능 + 내 실수를 개념 퀴즈로 가공해 학습을 완성시키는 에듀테크 지향.
Technology & Feasibility

무박 2일 내 핵심 데모(MVP) 완성을 위한
안정적인 초속성 개발 스택

⚡
동적 동기화 API 아키텍처
복잡한 깃허브 앱 심사 우회. GitHub REST API와 유저 개인 토큰(PAT) 연동으로 데모용 원클릭 수집.
🤖
Gemini JSON Mode 엔진
코드 코멘트 데이터셋을 완벽한 스키마의 퀴즈 데이터(JSON)로 자동 파싱하여 백엔드 복잡도를 최소화.
💎
Supabase + Recharts RPG 대시보드
Supabase Client-Side Direct Query로 초고속 DB 연동. Recharts로 개발자 성장을 상징하는 다이내믹 레이더(Radar) 차트 구현.
R&R & Milestone

팀원 6인의 유기적 역할 분담 및 마일스톤

  • 기획 및 디자인 (1명): RPG 비주얼 스킨 기획, 시연 영상 촬영/편집, 피칭 장표 제작.
  • AI / Prompt 엔지니어 (1명): Gemini 퀴즈 생성 및 자기설명(Explain-Back) 검증 프롬프트 튜닝.
  • 풀스택 엔지니어 (1명): Supabase 세팅, GitHub REST API 수집 모듈 구현.
  • 프론트엔드 엔지니어 (2명): RPG 스탯 대시보드(Radar Chart) 및 인터랙티브 퀴즈 UI 구현.
  • 데이터 코디네이터 (1명): 테스트용 과제 Boilerplate 및 가상 리뷰 시나리오 설계.
